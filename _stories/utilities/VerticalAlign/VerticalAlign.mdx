# Vertical Align

Adjust vertical alignment. Vertical alignment defaults to 'baseline'.

The available classes are:
* `-va-sub`
* `-va-super`
* `-va-top`
* `-va-middle`
* `-va-bottom`
* `-va-text-top`
* `-va-text-bottom`
