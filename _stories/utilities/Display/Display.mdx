# Display

Adjust the display property on any element. To adjust the property at specific breakpoints only, use the breakpoint-specific modifier classes.

The available classes are:
* `-block`
* `-inline-block`
* `-none`
* `-vis-hidden`
* `-sr-only`
* `-flex`

The breakpoint-specific modifier classes that can be added to any of the above classes are:
* `--xs`
* `--sm`
* `--md`
* `--lg`
* `--xl`

For example: `-block--xs`
