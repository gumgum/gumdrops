# ImagePreview

The `<ImagePreview>` component is used to preview images or gifs.

It will pre-load the image and show a loading indicator until resolved, then the preview will display.


## ImagePreview example
```jsx
<ImagePreview src="PATH_TO_IMAGE">
  <Icon icon="plus" />
</ImagePreview>
```
