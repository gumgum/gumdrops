# Overflow

Adjust the overflow property.

The available classes are:

-   `-overflow-hidden`
-   `-overflow-visible`
-   `-overflow-x-hidden`
-   `-overflow-y-hidden`
-   `-overflow-x-scroll`
-   `-overflow-y-scroll`
