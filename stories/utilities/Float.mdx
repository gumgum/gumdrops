# Float

Adjust the float property on any element, and clear the floats wherever needed.

The available classes for float are:

-   `-float-left`
-   `-float-right`
-   `-float-none`
-   `-float-none--xs`
-   `-float-none--sm`
-   `-float-none--md`
-   `-float-none--lg`
-   `-float-none--xl`

The available classes for clearing floats are:

-   `-clear-left`
-   `-clear-right`
-   `-clear-both`
