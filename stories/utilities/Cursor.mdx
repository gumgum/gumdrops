# Cursor

Adjust the cursor property.

The available classes are:

-   `-cursor--default`
-   `-cursor--pointer`
-   `-cursor--context-menu`
-   `-cursor--help`
-   `-cursor--progress`
-   `-cursor--crosshair`
-   `-cursor--text`
-   `-cursor--copy`
-   `-cursor--move`
-   `-cursor--not-allowed`
-   `-cursor--zoom-in`
-   `-cursor--zoom-out`
-   `-cursor--grab`
-   `-cursor--grabbing`
